# Cypress Tests - Knowde Academy

To run the test you need to install Cypress

## Instalation

```bash
npm install cypress
```

or visit https://www.cypress.io/, download and install

## Running tests

You need to copy tests to Cypress\tests\cypress\integration\tests folder.
